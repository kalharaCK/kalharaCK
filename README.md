## Some things I've built

**Analog Power Line Communication System**
Sending data over live AC mains using ASK/FSK modulation — modulator and demodulator built from discrete components, with galvanic isolation for safety.

**IIoT Industrial Machine Monitoring Node**
An ESP32 node with GSM connectivity for real-time machine health monitoring — thermocouples, IR sensors, current transformers, and a ThingsBoard dashboard with alerting, housed in an IP65 enclosure.

**Digital Multimeter on a Custom PCB**
An STM32-based multimeter with auto-ranging firmware, measuring voltage, current, and resistance, running roughly 68.5 hours on two 18650 cells.

**PCB Power-Up Circuit for an ARM9 MPU**
A compact, production-ready PCB with power sequencing and ESD protection — went on to mass production for the client.

**FPGA Stopwatch with Lap Timer**
An FSM-based stopwatch in SystemVerilog on the Nexys A7, built through the full Vivado flow from synthesis to bitstream.
